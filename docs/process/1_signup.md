# 报名
> 昭示决心

## 此课程面向哪些人：

- 想要制作自己的音乐
- 希望加深经验的职业音乐人
- 电台主播,up主,抖音达人,快手达人,vlogger
- 希望更了解自己天赋与局限的人
- 希望深入理解音乐及其他艺术领域的任何人
- 已经写了几首歌,但现在想了解更多有关音乐制作的信息的人


## 要求

- 理解并明确自己当前的渴望
- 每周都投入足够时间来探索
- 按时参加课程直播
